# morning_news_agent
Creation of a basic morning news agent that emails news relevant to topics you can specify.
Using the Python script attached, you can inut your email and specify topics that you want researched and emailed to you.
This uses NotebookLM which can be downloaded and set on your personal computer.
You must have Python 3.13 downloaded in order to automatically run
You also will need to edit the API that you are using. The code currently uses a free google/gemma-3-4B for the research, but changing this line to your chosen model and changing it to include your email and password would be a necessary step. 
Finally, you must set this up using Task Manager and set it up to your preferences.
